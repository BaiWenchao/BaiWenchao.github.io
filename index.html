博弈论01：简介与基本定义

发表于 2022-01-20 更新于 2022-01-21 分类于 学习

博弈分析框架

  • Players: 博弈的参与者是谁？
    • 人，公司，组织，政府，民族…
  • Actions: 博弈的参与者可以采取哪些行为？
    • 拍卖中的定价，是否决定罢工，何时出售股票，如何进行投票…
    • p.s. 参与者能够采取并对参与者的收益产生影响的行为都应该纳入考虑范围。
  • Timing of the Interactions: 博弈交互的时序？
    • 参与者的行为是同时发生的还是顺序发生的？
    • 参与者之间的交互持续发生吗？
    • 行为发生的顺序是什么？
    • 行为发生的顺序对参与者的收益有何影响？
    • 参与者采取行为时可以获得什么信息吗？
  • Payoffs: 博弈交互过程中的收益？
    • 收益的分析需要仔细研究参与者所有可能的行为。
    • 某些参与者（例如：自己）可能比其他参与者更加方便进行收益评估。
    • 分析参与者的行为决策策略是一个可能的难点。

博弈的一般形式（Normal Form）

  • Players (finite): $N = \{1,\cdots,n\}$
  • Actions:
    • action set (finite) for player $i$：$A_i$
    • set of action profiles of pure strategies or actions: $A = A_1\times\cdots\times A_n$
    • a generic action profile: $a=(a_1,\cdots,a_n)\in A$
  • Utility (or Payoff) function
    • utility function for player $i$: $u_i:A\rightarrow \mathbb{R}$
    • profile of utility functions: $u=(u_1,\cdots,u_n)$
